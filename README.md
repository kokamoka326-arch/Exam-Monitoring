 **Exam Proctoring System**
 
 This project is an **AI-Powered Exam Proctoring System** that monitors students during online exams in real time. The backend is built with **Node.js + Express + Socket.IO** for live communication, connected to a **SQL Server** database that stores exam sessions, cheat incidents, and face analysis data. The frontend consists of three portals — login, student exam room, and proctor dashboard — using **JWT-based ACL** to enforce role separation between students and proctors. Cheating detection covers tab switching, copy attempts, DevTools access, and **face-api.js** AI camera monitoring that flags multiple faces as violations.
